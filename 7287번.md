# 7287번: 등록

문제

자신이 백준 온라인 저지(BOJ)에서 맞은 문제의 수와 아이디를 그대로 출력하는 프로그램을 작성하시오.

입력

이 문제는 입력이 없다.

출력

첫 줄에 자신이 맞은 문제의 수, 둘째 줄에 아이디를 출력한다.



예제 입력 1 


예제 출력 1 
```
123
Your_ICPC_Team_Name
```

간단한 입출력 문제이다. 자신이 백준에서 **맞춘 문제 수와 아이디**를 그대로 출력하면 됩니다.

```swift
/// 그 당시 맞춘 문제 수
print("25")
print("kimsh153")
```
